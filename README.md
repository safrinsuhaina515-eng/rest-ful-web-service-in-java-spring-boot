# RESTful Web Service in Java Spring Boot

## Description
A RESTful web service developed using Spring Boot to handle HTTP requests and provide efficient API communication.

## Technologies Used
- Java
- Spring Boot
- Maven
- REST API
- HTML

## Features
- RESTful API development
- Handles HTTP GET requests
- Simple web interface
- Fast and lightweight application
- Easy deployment using Spring Boot

## Requirements
- Java JDK 17 or above
- Spring Boot
- Maven
- VS Code
- Web Browser

## Project Structure
- pom.xml
- src/main/java
- src/main/resources
- application.properties
- index.html

## How to Run
1. Clone the repository.
2. Open the project in VS Code.
3. Run the Spring Boot application.
4. Open your browser and visit:
   http://localhost:8080
